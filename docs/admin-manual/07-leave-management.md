# 07. 有給休暇管理

[< 目次に戻る](./00-index.md)

---

## 概要

スタッフの有給休暇の申請承認・残日数管理を行います。
管理画面 > **月次業務** > **有給休暇管理** から操作できます。

画面は3つのタブに分かれています。

---

## 申請一覧タブ

スタッフから提出された有給休暇の申請を確認し、承認・却下を行います。

### 申請の承認

1. 「申請一覧」タブを開きます（デフォルト表示）
2. 未処理の申請が一覧表示されます
3. 対象の申請の「**承認**」ボタンをクリックします
4. 必要に応じてコメントを入力します
5. 「承認する」をクリックして確定します

### 申請の却下

1. 対象の申請の「**却下**」ボタンをクリックします
2. 却下理由をコメント欄に入力します（任意）
3. 「却下する」をクリックして確定します

### 申請内容の確認

各申請には以下の情報が表示されます。

| 項目 | 説明 |
|------|------|
| スタッフ名 | 申請者 |
| 種別 | 全休 / 午前半休 / 午後半休 |
| 日付 | 取得希望日（複数可） |
| 理由 | 申請理由 |
| ステータス | 未処理 / 承認済 / 却下 / キャンセル |

> **補足**: 承認すると自動的にスタッフへ通知（お知らせ）が作成されます。

---

## 履歴タブ

過去の有給休暇申請の履歴を確認できます。

- ステータスでフィルタリングが可能です（未処理 / 承認済 / 却下 / キャンセル）
- 全ての申請履歴が時系列で表示されます

---

## 残日数管理タブ

各スタッフの有給休暇の残日数を管理します。

### 表示内容

| 項目 | 説明 |
|------|------|
| スタッフ名 | 対象者 |
| 入社日 | 有給付与の基準日 |
| 付与日数 | 自動計算された付与日数 |
| 繰越日数 | 前年度からの繰越 |
| 使用日数 | 今年度の使用済み日数 |
| 残日数 | 利用可能な残り日数 |

### 入社日の設定

1. 対象スタッフの入社日欄の「**設定**」ボタンをクリックします
2. 入社日を入力します
3. 「保存」をクリックします

> **重要**: 入社日を設定すると、勤続年数に基づいて有給休暇の付与日数が自動計算されます。

### 残日数の手動調整

自動計算では対応できないケースでは手動調整が可能です。

1. 対象スタッフの「**編集**」ボタンをクリックします
2. 調整日数を入力します（増加は正の値、減少は負の値）
3. 「保存」をクリックします
