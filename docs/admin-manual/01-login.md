# 01. ログイン・ログアウト

[< 目次に戻る](./00-index.md)

---

## ログイン

1. ブラウザで管理ポータル（`/admin`）にアクセスします
2. 「管理者パスワード」欄にパスワードを入力します
3. 「ログイン」ボタンをクリックします
4. 認証に成功すると、管理画面のメイン画面が表示されます

> **注意**: パスワードを連続で間違えるとロックされる場合があります。パスワードがわからない場合はシステム管理者に問い合わせてください。

## ログアウト

1. 画面右上の「ログアウト」ボタンをクリックします
2. ログイン画面に戻ります

## セッションについて

- ログイン状態は **7日間** 有効です
- 7日を過ぎると自動的にログアウトされます
- ブラウザを閉じてもセッションは維持されます
