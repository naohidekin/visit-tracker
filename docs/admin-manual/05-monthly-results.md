# 05. スタッフ月次実績

[< 目次に戻る](./00-index.md)

---

## 概要

各スタッフの月ごとの訪問実績を日別に確認・編集できます。
管理画面 > **月次業務** > **スタッフ月次実績** から操作できます。

---

## 実績の確認

1. 年・月を選択します
2. 対象のスタッフを選択します
3. 「**表示**」ボタンをクリックします

### 看護師の場合

以下のサマリーが表示されます。

| 項目 | 説明 |
|------|------|
| 介護時間 | 介護保険での訪問時間の合計 |
| 医療時間 | 医療保険での訪問時間の合計 |
| 合計時間 | 介護＋医療の合計 |
| 稼働日数 | 訪問を行った日数 |

その下に日別の詳細テーブルが表示され、各日の介護・医療の訪問時間が確認できます。

### リハビリスタッフ（PT/OT/ST）の場合

| 項目 | 説明 |
|------|------|
| 単位数 | 訪問単位数の合計 |
| 稼働日数 | 訪問を行った日数 |

---

## 実績の手動編集

日別テーブルで各日の値を直接編集できます。

1. 編集したい日のセルのドロップダウンを操作します
2. 値を選択します
3. 変更は自動的に保存されます

> **用途**: Googleスプレッドシートの値と齟齬がある場合の修正や、特殊な訪問記録の追加に使用します。
