# 03. お知らせ管理

[< 目次に戻る](./00-index.md)

---

## 概要

全スタッフに対するお知らせ（通知）を作成・編集・削除します。
管理画面 > **日常業務** > **お知らせ管理** から操作できます。

スタッフはログイン後のトップ画面やお知らせ画面でこれらの通知を確認できます。

---

## お知らせの作成

1. 「**新規作成**」ボタンをクリックします
2. 以下の項目を入力します

| 項目 | 説明 |
|------|------|
| 発信元 | 「管理者」または「システム」を選択 |
| タイトル | お知らせのタイトル |
| 本文 | お知らせの内容 |

3. 「**保存**」ボタンをクリックして投稿します

---

## お知らせの編集

1. 編集したいお知らせの「**編集**」ボタンをクリックします
2. 内容を修正します
3. 「**保存**」で確定します

---

## お知らせの削除

1. 削除したいお知らせの「**削除**」ボタンをクリックします
2. 確認ダイアログで「削除する」をクリックします

> **注意**: 削除したお知らせは復元できません。
