## 管理画面へのアクセス

管理ポータルのURL: `https://<アプリURL>/admin`

## 管理画面の構成

管理画面は以下の3つのカテゴリに分かれています。

| カテゴリ | 内容 |
|---------|------|
| **日常業務** | スタッフ一覧、お知らせ管理、未確定の予定 |
| **月次業務** | 月次実績、Excel集計、有給休暇、オンコール |
| **設定・マスタ** | インセンティブ設定、スタッフ追加、翌年シート作成 |

各カテゴリは折りたたみ式になっており、クリックで展開・折りたたみができます。
