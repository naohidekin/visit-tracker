# 08. オンコール管理

[< 目次に戻る](./00-index.md)

---

## 概要

スタッフのオンコール当番の実績を月別に確認できます。
管理画面 > **月次業務** > **オンコール管理** から操作できます。

---

## 月次サマリーの確認

1. 対象の **年月** を選択します
2. スタッフごとのオンコール実績サマリーが表示されます

### サマリー項目

| 項目 | 説明 |
|------|------|
| 回数 | オンコール対応の回数 |
| 合計時間 | 対応にかかった合計時間（分） |
| 搬送回数 | 搬送が発生した回数 |
| 当番日数 | オンコール当番を担当した日数 |

---

## 詳細記録の確認

サマリーからスタッフを選択すると、日別の詳細記録を確認できます。

各記録には以下の情報が含まれます。

| 項目 | 説明 |
|------|------|
| 日付 | オンコール対応日 |
| 対応回数 | その日の対応回数 |
| 対応時間 | 合計対応時間（分） |
| 搬送有無 | 搬送が発生したかどうか |

> **注意**: オンコール記録の入力はスタッフ自身が行います。管理画面では閲覧のみです。
