# 08. オンコール管理

[< 目次に戻る](./00-index.md)

---

## 概要

スタッフのオンコール当番の実績を月別に確認できます。
管理画面 > **月次業務** > **オンコール管理** から操作できます。

---

## 月次サマリーの確認

1. 対象の **年月** を選択します
2. スタッフごとのオンコール実績サマリーが表示されます

### サマリー項目

| 項目 | 説明 |
|------|------|
| 件数 | オンコール対応の件数 |
| 時間 | 対応にかかった合計時間 |
| 交通費 | 交通費が発生した件数 |
| 日数 | オンコールを担当した日数 |

> **注意**: オンコール記録の入力はスタッフ自身が行います。管理画面では閲覧のみです。
