# 10. スタッフ追加（詳細）

[< 目次に戻る](./00-index.md)

---

## 概要

新しいスタッフをシステムに追加します。
管理画面 > **設定・マスタ** > **スタッフ追加** から操作できます。

---

## 追加手順

### 1. 基本情報の入力

| 項目 | 必須 | 説明 |
|------|------|------|
| 姓（漢字） | はい | 例: 田中 |
| 名（漢字） | はい | 例: 太郎 |
| 姓（ふりがな） | はい | 例: たなか |
| 名（ふりがな） | はい | 例: たろう |
| 職種 | はい | 看護師 / PT / OT / ST から選択 |
| 入社日 | いいえ | 有給休暇の自動計算に使用 |
| オンコール対象 | いいえ | オンコール当番の対象にするか |

### 2. 自動生成される情報

入力されたふりがなをもとに、以下が自動生成されます。

| 項目 | 説明 |
|------|------|
| ログインID | ふりがなのローマ字表記 + 連番 |
| 初期パスワード | ふりがなのローマ字表記の一部 |

### 3. 確認・登録

1. 入力内容を確認します
2. 「**追加**」ボタンをクリックします
3. 登録完了のメッセージと共に、ログインID・初期パスワードが表示されます

> **重要**: 表示されたログインID・初期パスワードは必ず控えてスタッフに伝えてください。初期パスワードは後から確認できません。

---

## 登録時の自動処理

スタッフ追加時に、以下の処理が自動で行われます。

1. **スタッフ情報の保存**: staff.json に新規エントリを追加
2. **Googleスプレッドシートへの列追加**:
   - 看護師: 2列（介護・医療）を追加
   - リハビリスタッフ: 1列を追加
   - 当年・翌年のすべての月シートに列が追加されます
3. **パスワードのハッシュ化**: bcryptで暗号化して保存

---

## 注意事項

- 同じふりがなのスタッフが既に存在する場合、ログインIDの末尾に連番が付きます
- 追加後に職種を変更することはできません
- 入社日は後からでも設定可能です（有給休暇管理画面から）
