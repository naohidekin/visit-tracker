ログイン・Face ID

ID・パスワードでログイン
1
ログインIDとパスワードを入力
管理者から伝えられたログインIDとパスワードを入力し、「ログイン」をタップします。
2
初回は初期パスワードでログイン
初回ログイン後、「PW変更」からパスワードを変更することをおすすめします。
Face ID（生体認証）でログイン
1
Face IDを事前に登録
「PW変更」画面で Face ID を登録すると、次回から生体認証だけでログインできます。
2
ログイン画面で「Face ID でログイン」をタップ
ログインIDを入力し、Face ID ボタンをタップすると、顔認証・指紋認証でログインできます。

Face ID を登録済みの場合、ログイン画面に自動で Face ID ボタンが表示されます。

パスワードを忘れた場合
ログイン画面下部の「パスワードを忘れた方はこちら」からメールアドレスを入力してリセットを申請できます。メールアドレスが未登録の場合は管理者にご連絡ください。

基本の流れ

1
訪問日を選択
トップ画面で訪問した日付を選びます。当日・翌日・過去の修正可能な日が選べます。
2
時間 / 単位を入力
看護師は「介護」「医療」の合計訪問時間を、リハビリは訪問単位数を入力します。
3
「記録する」をタップ
入力内容が実績に記録されます。完了画面に月の累計実績とインセンティブ情報が表示されます。

記録後に「続けて入力する」をタップすると、別の日の入力に移れます。

💉

看護師の入力方法

「介護」「医療」それぞれの合計訪問時間（時間単位）を選択します。
訪問時間入力する値
30分0.5
60分1.0
90分1.5

例）合計390分の場合は6.5時間と入力

該当する訪問がない場合（例：その日は医療訪問なし）は「空白」のままでOKです。

🏃

リハビリの入力方法

「介護」は10分を「1単位」として単位数を入力します。
「医療」は1件「4単位」として計算してください。

例）介護280分 + 医療1件の場合：28単位 + 4単位 ＝ 32単位と入力

📝

予定の登録・確定

翌日の予定訪問件数が決まっている場合、前日のうちに「予定」として登録できます。
1
翌日の日付を選んで入力
日付ピッカーで翌日を選択し、通常どおり時間や単位を入力して「記録する」をタップします。翌日のデータは自動的に「予定」として保存されます。
2
翌日になったら「確定」をタップ
トップ画面の「未確定の予定」カードに保存された予定が表示されます。内容を確認して「確定」ボタンをタップしてください。

確定するまで実績には反映されません。忘れずに確定してください。

予定が変わった場合は「削除」して入力し直すこともできます。

📊

月次実績の確認

メニューの「月次実績」から、月ごとの訪問記録を確認できます。
1
月を選択
画面上部のドロップダウンで確認したい月を選びます。
2
月間サマリーを確認
合計時間（介護・医療の内訳）、稼働日数、インセンティブ目安時間・単位が表示されます。
3
グラフ・日別一覧を展開
「グラフを見る」「日別一覧」をタップすると、日ごとの訪問量をグラフや表で確認できます。修正可能な日はここから直接編集もできます。
インセンティブ表示について
インセンティブ目安に到達すると、実績カードに到達メッセージが表示されます。インセンティブラインとの差もグラフ上に表示されるので、月の途中での進捗確認にも活用できます。
📅

締日・修正ルール

!
毎月15日が締日です
!
前月分の修正は当月20日まで
前月16日〜当月15日までのデータは当月20日まで修正できます。21日になると前月分は全てロックされ修正できなくなります。
!
当月分（16日以降）はいつでも修正可能
当月16日以降に入力したデータは、翌月20日まで修正できます。

毎月16〜20日に修正可能期間のお知らせが届きます。修正がある方はこの期間内にお願いします。

期限を過ぎた場合は管理者にご連絡ください。

🌴

有給管理

メニューの「有給管理」から、有給残日数の確認や休暇申請ができます。
残日数の確認
画面上部に現在の有給残日数が大きく表示されます。内訳として年間付与・前年繰越・使用済み日数なども確認できます。
休暇を申請する
1
休暇の種類を選択
全日（1日）、午前半休、午後半休から選べます。お祝い休暇が残っている場合は「お祝い休暇」も選択できます。
2
日付を指定
開始日を選択します。複数日にまたがる場合は終了日も選択してください。
3
理由を入力して申請
必要に応じて理由を入力し、「申請する」をタップします。管理者の承認後に確定します。
申請の取り消し
申請履歴から「保留中」「承認済み」の申請を取り消すことができます。

午前半休と午後半休はそれぞれ0.5日としてカウントされます。同じ日に午前・午後の両方を申請することも可能です。

有給付与のしくみ
勤続年数付与日数
6ヶ月10日
1年6ヶ月12日
2年6ヶ月14日
3年6ヶ月16日
4年6ヶ月18日
5年6ヶ月〜20日

前年の未使用分は翌年に繰り越されます（1年間有効）。次回付与日や繰越期限は有給管理画面で確認できます。

お祝い休暇
入職から6ヶ月以内に使える特別休暇です。期限を過ぎると使用できなくなります。残日数は有給管理画面で確認できます。

オンコール記録

オンコール対象のスタッフはメニューに「オンコール」が表示されます。当番の記録を入力・管理できます。
1
日付と合計時間を入力
オンコール対応を行った日付と合計対応時間を入力します。
2
交通費回数を入力
交通費が発生した回数を選択します。
3
「登録する」をタップ
記録が保存されます。同じ日の記録は上書きされます。
月間サマリー
画面上部に月の合計時間・交通費回数・担当日数が表示されます。月を切り替えて過去の記録も確認できます。
オンコール代休について
累計対応時間が15時間に達するごとに、1日の代休が自動で付与されます。進捗はゲージで確認できます。

付与された代休日数は有給管理画面の「累積OC付与」にも反映されます。

📢

お知らせ

管理者からのお知らせやシステム通知を確認できます。
1
未読のお知らせを確認
未読のお知らせがある場合、メニューのお知らせ横に赤いバッジが表示されます。タイトルの横に青い丸が表示されているのが未読です。
2
タップして内容を表示
お知らせをタップすると詳細が開き、自動的に既読になります。

修正可能期間のお知らせや未入力のリマインダーなど、システムからの自動通知もここに届きます。

パスワード変更・Face ID設定

パスワードを変更する
1
メニューから「PW変更」を選択
右上のハンバーガーメニューから「PW変更」をタップします。
2
現在のパスワードと新しいパスワードを入力
パスワードは4〜20文字で設定してください。確認のため新しいパスワードは2回入力します。
Face ID を設定する
パスワード変更画面の上部に「Face ID」セクションがあります。
1
「Face ID を登録する」をタップ
端末の生体認証（顔認証・指紋認証）の画面が表示されます。
2
認証を完了する
登録が完了すると、次回から Face ID でログインできます。

Face ID は端末ごとに登録が必要です。スマートフォンとPCなど、複数端末で使う場合はそれぞれで登録してください。

Face ID に対応していない端末では、この機能は表示されません。

ホーム画面に追加

このアプリはホーム画面に追加すると、アプリのように使えます。
iPhone（Safari）の場合
1
Safariで画面下の共有ボタンをタップ
四角に上矢印のアイコンをタップします。
2
「ホーム画面に追加」を選択
名前を確認して「追加」をタップすれば完了です。
Android（Chrome）の場合
1
Chromeの右上メニューをタップ
「︙」アイコンをタップします。
2
「ホーム画面に追加」を選択
名前を確認して「追加」をタップすれば完了です。

ホーム画面から起動すると、ブラウザのアドレスバーなしの全画面表示になり、より使いやすくなります。
