🖥️ 画面の構成

▼

管理画面はPC向けの サイドバー＋メインコンテンツ レイアウトです。

サイドバー（左側メニュー）

左側のメニューから各機能にアクセスします。メニューは3つのグループに分かれています。

グループメニュー項目
日常業務ダッシュボード / 未確定の予定 / スタッフ一覧 / お知らせ管理 / 有給休暇管理 / 操作履歴
月次業務待機・雨の日 / オンコール / インセンティブ集計 / スタッフ月次実績 / iBow Excel集計 / iBow集計履歴 / 出勤確定
設定・マスタインセンティブ設定 / スタッフ追加 / 翌年スプレッドシート
モバイル表示

スマートフォンではサイドバーは非表示になり、左上の ☰ ボタンで開閉できます。

ダッシュボード

ログイン直後に表示される画面です。以下の情報がカードで表示されます。

  • 未確定の予定 — 処理待ちの予定件数（クリックで一覧へ）
  • 未処理の有給申請 — 承認待ちの件数（クリックで一覧へ）
  • スタッフ数 — 有効なスタッフの人数（クリックで一覧へ）

各カードをクリックすると該当セクションに直接ジャンプします。

🔐 ログイン・ログアウト

▼

管理画面は 個別アカウント制 です。管理者権限を持つスタッフIDでログインします。

Face IDでログイン（推奨）

Face ID（生体認証）が登録済みの場合、自動的にFace IDログインが表示されます。

  1. /admin にアクセス
  2. ログインIDを入力（Enter or フォーカス移動で自動判定）
  3. 「Face IDでログイン」ボタンが表示される
  4. ボタンをクリック → 生体認証を実行
  5. 認証成功でダッシュボードが表示される
ポイント: Face IDはパスワード入力が不要で、最も安全かつ簡単なログイン方法です。
パスワードでログイン

Face ID未登録の場合、またはFace ID画面で「パスワードでログイン」をクリックした場合。

  1. ログインIDを入力
  2. 「パスワードでログイン」リンクをクリック
  3. パスワードを入力（8文字以上）
  4. 「ログイン」をクリック
注意: パスワードは8文字以上が必要です。短いパスワードの場合は先にパスワード変更が必要です。
緊急ログイン（共有パスワード）

個別アカウントでログインできない緊急時に、共有パスワードでログインできます。

  1. ログイン画面下部の「緊急ログイン（共有パスワード）」をクリック
  2. 共有パスワードを入力
  3. 「ログイン」をクリック
重要: 緊急ログインは操作履歴に記録されます。通常は個別アカウントをご利用ください。
セッションの自動復元
  • セッションは 7日間 有効です。
  • ブラウザを閉じても維持されます。
  • 有効なセッションがある場合、ページを開くと自動的にログイン状態が復元されます。
ログアウト

サイドバー下部の「ログアウト」ボタンをクリックします。全セッション情報が完全に破棄されます。

アーカイブ・復元・削除
操作手順説明

オンコール対象の切り替え（看護師のみ）

「OC有」「OC無」ボタンで切り替えます。対象に設定されたスタッフのみ、オンコール記録の入力が可能になります。

管理者権限の付与・剥奪

「管理者」ボタンで管理者権限を付与・剥奪できます。管理者に設定されたスタッフは管理画面にログインできます。

📢 お知らせ管理

▼
日常業務 > お知らせ管理

スタッフはトップ画面およびお知らせ画面で確認できます。

お知らせの作成
  1. 発信元を選択（管理者 / 運営）
  2. タイトルを入力
  3. 本文を入力
  4. 「新規作成」をクリック
編集・削除
  • 一覧の「編集」ボタンで内容を修正できます。
  • 「削除」ボタンで削除できます（復元不可）。
自動配信されるお知らせ
  • 修正可能期間通知 — 毎月16日に自動配信（16日〜20日の修正期間を案内）
  • 未入力リマインダー — 平日18時に未入力のスタッフへ個別通知
  • アップデート情報 — システム更新時に自動配信

📋 未確定の予定管理

▼
日常業務 > 未確定の予定一覧

スタッフが入力した未確定の予定（翌日以降の訪問予定）を確認・削除できます。

  • スタッフ名・日付・職種・入力値・登録日時が表示されます。
  • 「削除」ボタンで不要な予定を削除できます。
重要: 削除は復元できません。スタッフに確認の上で操作してください。

🏖️ 有給休暇管理

▼
日常業務 > 有給休暇管理

3つのタブで構成されています: 申請一覧・履歴・残日数管理

申請一覧タブ

承認待ちの有給申請が表示されます。

承認する場合:

  1. 「承認」ボタンをクリック
  2. コメントを入力（任意）
  3. 「承認する」をクリック

却下する場合:

  1. 「却下」ボタンをクリック
  2. コメントを入力（任意）
  3. 「却下する」をクリック

処理結果はスタッフのお知らせに自動通知されます。

履歴タブ
  • スタッフ名やステータス（承認待ち / 承認済 / 却下 / 取消済）でフィルタリングできます。
  • CSV出力が可能です。
残日数管理タブ

各スタッフの有給残日数を管理します。

項目説明
入社日自動付与の基準日。未設定の場合は手動付与が必要。
付与日数勤続年数に基づき自動計算（6ヶ月:10日〜6.5年:20日）
繰越日数前年度からの繰越（最大1年分）
手動調整正の値=増加、負の値=減少
残日数付与+繰越+調整-使用 で自動計算

残日数の編集:

  1. 「編集」ボタンをクリック
  2. 入社日・付与日数・繰越日数・手動調整を入力
  3. 「保存」をクリック

📝 操作履歴

▼
日常業務 > 操作履歴

すべての管理操作がタイムスタンプ付きで記録されます。SHA-256ハッシュチェーンにより改ざんを検知できます。

フィルタリング
  • 操作種別: 認証 / 実績入力 / 予定 / スタッフ管理 / 有給 / オンコール / お知らせ / インセンティブ
  • 日付範囲: 開始日〜終了日で絞り込み
整合性チェック

「整合性チェック」ボタンで、ログが改ざんされていないか検証できます。

CSV出力

フィルタリング結果をCSVでダウンロードできます。

🌧️ 待機・雨の日管理

▼
月次業務 > 待機・雨の日

毎月16日〜翌月15日の待機当番と雨の日手当を管理します。

待機当番の登録
  1. 月を選択（支払月ベース: 4月 = 3/16〜4/15）
  2. 各日付の「待機者」ドロップダウンでスタッフを選択
  3. 選択と同時に自動保存されます
待機料の自動計算
区分待機料
平日2,000円
土曜5,000円
日曜・祝日10,000円
雨の日の登録

各日付の右端にある ✅/⬜ アイコンをクリックして、雨の日を登録・解除します。

  • ✅ = 雨の日として登録済み
  • ⬜ = 未登録

雨の日に出勤したスタッフには手当が発生します。集計は画面下部に自動表示されます。

カスタム祝日の追加

「祝日設定」ボタンから、年末年始など国民の祝日以外の休日を追加できます。追加した日は祝日扱い（待機料10,000円）になります。

CSV出力

「CSV出力」ボタンで待機一覧をダウンロードできます。

  • 休暇（leave）: 有給申請が承認済み → 休暇として自動確定

💰 インセンティブ設定

▼
設定・マスタ > インセンティブ設定
デフォルト基準の設定
  1. 看護師: 1日あたりの基準時間を選択（例: 3.5時間）
  2. リハビリ: 1日あたりの基準単位を選択（例: 20単位）
  3. 「デフォルトを保存」をクリック

基準を超過した月にインセンティブが発生します。

個別設定

スタッフごとにデフォルトと異なる基準値を設定できます。

  1. 各スタッフのライン入力欄に値を入力
  2. 「保存」をクリック
  • 時短勤務者の「勤務時間（h/日）」も設定可能
  • 個別設定がある場合、デフォルト基準より優先されます

📅 翌年スプレッドシート作成

▼
設定・マスタ > 翌年スプレッドシート
作成手順
  1. 「翌年シートを作成」ボタンをクリック
  2. 処理完了後、スプレッドシートのURLが表示される
作成される内容
  • 12枚の月別シート（1月〜12月）

注意: 同じ年のシートが既に存在する場合はエラーになります。
