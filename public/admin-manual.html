🔐 ログイン・ログアウト

▼
ログイン手順
  1. /admin にアクセス
  2. パスワードを入力
  3. ログインボタンをクリック
  4. メイン画面が表示される
注意: パスワードがわからない場合はシステム管理者に問い合わせてください。
ログアウト

画面右上の「ログアウト」ボタンをクリックします。

セッションについて
  • セッションは 7日間 有効です。
  • ブラウザを閉じても維持されます。

👥 スタッフ管理

▼

場所: 日常業務 > スタッフ一覧

一覧表示
  • ふりがな順に並んでいます。
  • 名前・職種・ログインIDが表示されます。
  • 「アーカイブ済みも表示」で退職者の表示を切り替えられます。
職種バッジ
職種コード表示名
看護看護師
PT理学療法士
OT作業療法士
ST言語聴覚士
スタッフ情報の編集
  1. 「編集」ボタンをクリック
  2. 名前・ふりがなを修正
  3. 「保存」または「取消」をクリック
注意: ログインID・職種は編集できません。
パスワードリセット
  1. 「PW」ボタンをクリック
  2. 確認ダイアログで「リセット実行」をクリック
  3. 初期パスワードが表示される
  4. 表示されたパスワードをスタッフに伝える
注意: リセット後、スタッフにパスワードを変更してもらってください。
アーカイブ
  1. 「アーカイブ」ボタンをクリック
  2. 確認ダイアログで承認
  3. スタッフが一覧から非表示になる
復元
  1. 「アーカイブ済みも表示」をクリック
  2. 該当スタッフの「復元」ボタンをクリック
削除
  • アーカイブ済みのスタッフのみ削除できます。
  • 「削除」ボタン → 確認ダイアログで実行します。
重要: 削除は元に戻せません。通常はアーカイブで十分です。
オンコール対象（看護師のみ）
  • 「OC有」「OC無」ボタンをタップして切り替えます。
  • オンコール対象に設定されたスタッフのみ、オンコール入力が可能になります。

📢 お知らせ管理

▼

場所: 日常業務 > お知らせ管理

スタッフはトップ画面やお知らせ画面で確認できます。

お知らせの作成
  1. 「新規作成」をクリック
  2. 発信元（管理者/システム）を選択
  3. タイトル・本文を入力
  4. 「保存」をクリック

  1. 「編集」ボタンをクリック
  2. 内容を修正
  3. 「保存」をクリック
お知らせの削除
  1. 「削除」ボタンをクリック
  2. 確認ダイアログで承認
重要: 削除したお知らせは復元できません。

🏖️ 有給休暇管理

▼

場所: 月次業務 > 有給休暇管理

3つのタブで構成されています。

申請一覧タブ

申請情報として以下が表示されます:

  • スタッフ名
  • 種別（全休 / 午前半休 / 午後半休）
  • 日付
  • 理由
  • ステータス（未処理 / 承認済 / 却下 / キャンセル）

承認する場合:

  1. 「承認」ボタンをクリック
  2. コメントを入力（任意）
  3. 「承認する」をクリック

却下する場合:

  1. 「却下」ボタンをクリック
  2. コメントを入力（任意）
  3. 「却下する」をクリック

承認すると、スタッフに自動通知されます。

履歴タブ
  • ステータスでフィルタリングできます。
  • 時系列で表示されます。
残日数管理タブ

以下の情報が表示されます:

  • スタッフ名・入社日・付与日数・繰越日数・使用日数・残日数

入社日の設定:

  1. 「設定」ボタンをクリック
  2. 日付を入力
  3. 「保存」をクリック

勤続年数に基づき、付与日数が自動計算されます。

手動調整:

  1. 「編集」ボタンをクリック
  2. 調整日数を入力（正の値 = 増加、負の値 = 減少）
  3. 「保存」をクリック

💰 インセンティブ設定

▼

場所: 設定・マスタ > インセンティブ設定

デフォルト基準
  • 看護師: 1日あたり基準時間（例: 3.5h）
  • リハビリ: 1日あたり基準単位（例: 20）

基準を超過した場合にインセンティブ対象となります。

設定手順
  1. 看護師の基準時間を入力
  2. リハビリの基準単位を入力
  3. 「保存」をクリック
個別設定
  1. スタッフを選択
  2. 個別の値を入力
  3. 「保存」をクリック
注意: 個別設定がある場合、デフォルト基準は適用されません。

▼

入力項目
項目必須説明
姓（漢字）必須

名（ふりがな）必須

入社日任意後から設定可能

追加手順
  1. 各項目を入力
  2. 「追加」をクリック
  3. 完了メッセージとログインID・初期パスワードが表示される
重要: 表示されたログインIDと初期パスワードを控えて、スタッフに伝えてください。
自動生成される情報
  • ログインID: ローマ字 + 連番で自動生成
  • 初期パスワード: ローマ字の一部から自動生成
自動処理
  • staff.jsonに保存
  • スプレッドシートに列を追加（看護師: 2列 / リハビリ: 1列）
  • パスワードは暗号化して保存
注意: 同名のふりがなの場合は連番が付与されます。職種は後から変更できません。入社日は後から設定可能です。

📅 翌年スプレッドシート作成

▼

場所: 設定・マスタ > 翌年スプレッドシート作成

作成手順
  1. 「翌年スプレッドシート作成」ボタンをクリック
  2. 確認ダイアログで承認
  3. 作成されたスプレッドシートのURLが表示される
作成される内容
  • 12枚の月別シート
  • スタッフ列（看護師: 2列 / リハビリ: 1列）
  • ヘッダー情報
管理情報
  • spreadsheet-registry.jsonに記録されます。
  • 年度ごとに自動参照されます。
推奨: 年末（12月頃）に実行してください。
注意: 既存のスプレッドシートがある場合は、別のシートが生成されます。追加されたスタッフは自動的に反映されます。
重要: Google共有設定は別途必要です。
